    🌐 Browser Rendering Pipeline

    HTML

    Browser downloads HTML document.

    ➜

    Tokenization

    Converts raw HTML text into tokens such as tags, attributes and text nodes.

    ➜

    Parsing

    Browser analyzes tokens and understands the document structure.

    ➜

    DOM Tree

    Tree representation of all HTML elements.

    CSS

    Browser loads stylesheets

    ➜

    Tokenization

    Converts CSS text into selectors, properties and values.

    ➜

    Parsing

    Creates a structured stylesheet model.

    ➜

    CSSOM

    CSS Object Model containing all styles.

    JavaScript

    Browser downloads JS

    ➜

    Tokenization

    Breaks code into keywords, identifiers and operators.

    ➜

    Parsing

    Creates AST (Abstract Syntax Tree).

    ➜

    AST

    Structured representation of code.

    ⬇

    Render Tree

    DOM + CSSOM combine together. Hidden elements (display:none) are excluded.

    ⬇

    Layout (Reflow)

    Calculates width, height and position of every visible element.

    ⬇

    Paint

    Converts layout information into pixels. Draws text, colors, borders and images.

    ⬇

    Compositing

    Combines multiple layers using the GPU for better performance.

    ⬇

    Pixels On Screen

    Final rendered webpage visible to the user.
